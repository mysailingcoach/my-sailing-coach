### Pricing Comparison

#### My Free Sailing Coach
- **Cost**: Free
- **Hosting**: Self-hosted or cloud
- **Limitations**: None (storage limited by disk)
- **Premium**: N/A

#### ChartedSails
- **Free Plan**: Limited races, basic features
- **Pro**: ~$5/month, unlimited races, advanced analysis
- **Plus**: ~$10/month, social features, exports
- **Hosting**: Cloud only

#### Raceqs.com
- **Free Plan**: Limited races, basic features
- **Standard**: ~$4/month, unlimited races
- **Premium**: ~$8/month, all features, fleet management
- **Hosting**: Cloud only

## Integration Options

### Use MFSC With Other Tools

**Strava**: Export from Strava → Upload to MFSC  
**TrainingPeaks**: Download GPX → Upload to MFSC  
**Garmin Connect**: Export from Garmin → Upload to MFSC  
**Apple Health**: Export from Apple Health → Upload to MFSC  

### Export From MFSC

**Current**: Download from browser, manual export  
**Planned**: 
- Direct GPS export
- PDF reports
- CSV data export
- Integration APIs
